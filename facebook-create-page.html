Crea la Pagina Facebook

Il profilo Bitcoin.Wizard1 non supporta i post automatici via API. Serve una Pagina Facebook separata.

Apri la creazione pagina

Crea Pagina Facebook Business Manager → Pagine

Devi essere loggato con lo stesso account Facebook che usi per The Little Satoshi News.

Passo 1 — Tipo e nome

  1. Clicca Crea Pagina Facebook sopra
  2. Tipo: Azienda o marchio (oppure Creatore di contenuti)
  3. Nome pagina: copia qui sotto
  4. Categoria: Sito web educativo o Istruzione

Passo 2 — Foto profilo e copertina

Scarica foto profilo · Scarica copertina

Passo 3 — Bio e descrizione

Passo 4 — Collega al Business Manager

  1. Apri business.facebook.com
  2. Portfolio: The Little Satoshi News Poster
  3. Impostazioni → Pagine → Aggiungi → seleziona la nuova pagina The Little Satoshi News
  4. Assegna a te il controllo completo (Admin)

Passo 5 — Dopo la creazione (API)

  1. Apri Graph API Explorer
  2. Generate Access Token → spunta pages_manage_posts, pages_read_engagement, pages_show_list
  3. Query GET me/accounts → deve comparire la nuova pagina con id e access_token
  4. Copia l'access_token della pagina (non il User Token)
  5. Esegui: python scripts\aggiorna-token-facebook.py TOKEN
  6. Aggiorna FACEBOOK_PAGE_ID in scripts\.env con l'id numerico della pagina

Quando hai l'URL della nuova pagina (es. facebook.com/cryptofacile), mandamelo: aggiorno il sito e attivo i post automatici.
