Fix post Facebook

Pagina: The Little Satoshi News

Problema trovato: il token Facebook è scaduto il 19 giugno 2026. Da allora nessun post automatico (né immagini) viene pubblicato.
Le immagini sul sito funzionano (URL pubblici su GitHub Pages). Il problema non è il sito — serve un nuovo token Pagina.

Rinnova il token (5 minuti)

  1. Apri Graph API Explorer
  2. Generate Access Token → spunta pages_manage_posts, pages_read_engagement, pages_show_list
  3. Seleziona la pagina The Little Satoshi News
  4. Sotto il token: menu User or Page → scegli la PAGINA (non User Token)
  5. Copia il token (inizia con EAA...)
  6. Esegui: 
    python scripts\aggiorna-token-facebook.py INCOLLA_TOKEN_QUI

Test post con immagine

python scripts\verify-facebook-token.py
python scripts\post-to-facebook.py --auto --slot 0 --dry-run
python scripts\post-to-facebook.py --auto --slot 0
