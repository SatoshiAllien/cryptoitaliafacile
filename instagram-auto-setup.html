Post automatici su Instagram

20 post al giorno (07:00–22:00 Roma) su @bitcoin.is.hope2030 — immagini + caption in inglese, link in bio.

Cosa fa il sistema

  • 20 post/giorno — circa uno ogni 47 minuti
  • Immagini quadrate 1080×1080 da assets/img/instagram/posts/
  • Contenuti educativi crypto da data/articles.json
  • Automazione GitHub Actions (PC spento) o Windows Task Scheduler

Passo 1 — Account Instagram Business

  1. Apri @bitcoin.is.hope2030
  2. Impostazioni → Account → Passa a account professionale
  3. Scegli Business o Creator
  4. Collega a una Facebook Page (es. Crypto Italia Facile)

Senza collegamento a una Facebook Page, l'API Instagram non può pubblicare.

Passo 2 — Permessi Meta App

  1. Casi d'uso app
  2. Aggiungi Instagram API (o “Gestisci messaggi e contenuti su Instagram”)
  3. Permessi richiesti:
    • instagram_basic
    • instagram_content_publish
    • pages_read_engagement
  4. Rigenera il Page Access Token nel Graph API Explorer

Passo 3 — Configura .env

python scripts\discover-instagram-id.py
python scripts\verify-instagram.py

Aggiunge automaticamente INSTAGRAM_ACCOUNT_ID in scripts/.env.

Passo 4 — Genera immagini e test

python scripts\generate-ig-images.py
python scripts\post-to-instagram.py --auto --slot 0 --dry-run
python scripts\post-to-instagram.py --auto --slot 0

Passo 5 — Automazione cloud

powershell -ExecutionPolicy Bypass -File scripts\setup-instagram-github-secrets.ps1
powershell -ExecutionPolicy Bypass -File scripts\register-instagram-tasks.ps1
