Build di Computer Avanzati

Progetto e assemblo PC ad alte prestazioni ottimizzati per workload specifici: workstation AI, gaming competitivo, sviluppo software e ambienti di virtualizzazione. Ogni build parte dall'analisi del caso d'uso — non si monta hardware a caso, si progetta un sistema equilibrato.

Curo airflow, cable management e dissipazione termica con la stessa attenzione riservata ai componenti: flussi d'aria calcolati, radiatori posizionati strategicamente, paste termiche di qualità e curve ventole tarate su profili silenziosi o performance.

Configuro BIOS/UEFI avanzato: XMP/EXPO per la RAM, PBO per Ryzen, impostazioni PCIe, virtualizzazione (SVM/VT‑x) e avvio sicuro. Ogni workstation è pronta per AI inference, container, VM e sessioni di sviluppo prolungate.

  • AMD Ryzen
  • NVIDIA GPU
  • SSD NVMe
  • RAM Tuning
  • Custom Loop

Installazione e Gestione Sistemi Operativi

Installo e configuro Windows 11 Pro, Linux/Ubuntu e macOS su hardware custom e laptop, adattando ogni sistema alle esigenze del workload. Gestisco setup multi‑boot, partizioni GPT, crittografia disco e policy di sicurezza di base.

Utilizzo quotidianamente PowerShell, Terminale Linux, Bash e Zsh per automazioni, deploy e manutenzione. Ho esperienza con WSL2 come ponte Windows‑Linux, containerizzazione leggera e script di provisioning per replicare ambienti in pochi minuti.

Il troubleshooting è parte del processo: driver, rete, kernel, repository e pacchetti — ogni problema è un'occasione per capire come funziona il sistema sotto il cofano.

  • Windows 11 Pro
  • Ubuntu / Linux
  • WSL2
  • Multi‑boot
  • Automazione

AI Locali & Modelli LLM

Installo e gestisco modelli AI locali — Llama, Qwen, Mistral, Phi e altri — sfruttando Ollama, WSL e GPU NVIDIA per l'accelerazione hardware. Creo ambienti AI personalizzati con modelli quantizzati, context window ottimizzati e API locali su 127.0.0.1.

Sperimento con prompt engineering avanzato, agenti AI autonomi e automazioni che collegano LLM locali a workflow operativi. Il server AI gira in casa, senza dipendere da cloud esterni — privacy, controllo e latenza minima.

Questa competenza alimenta direttamente progetti come Satoshi AI, l'assistente integrato in CryptoItaliaFacile.

  • Ollama
  • Llama / Qwen / Mistral
  • GPU CUDA
  • Agenti AI
  • 127.0.0.1

Terminale & Scripting

Il terminale è il mio strumento quotidiano. Con PowerShell automatizzo installazioni, backup, monitoraggio sistema e deploy di configurazioni. Su Linux uso comandi avanzati per networking, permessi, gestione pacchetti e processi.

Gestisco file system, mount point, tunnel SSH e configurazioni remote su server e workstation. Scrivo script per ottimizzare workflow personali — dalla sincronizzazione file al provisioning di ambienti di sviluppo completi in un solo comando.

Ogni script è documentato, modulare e pensato per essere riutilizzato: automazione non è solo comodità, è ripetibilità e affidabilità.

  • PowerShell
  • Bash / Zsh
  • SSH
  • Automazione
  • DevOps leggero
