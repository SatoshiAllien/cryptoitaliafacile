1. Get X API keys

  1. Go to developer.x.com
  2. Create a project + app with Read and Write permissions
  3. Generate keys while logged in as @TheRiser100x
  4. Copy: API Key, API Secret, Access Token, Access Token Secret

Shortcut: run scripts\apri-x-developer.bat

2. Add keys to scripts/.env

X_API_KEY=your_api_key
X_API_SECRET=your_api_secret
X_ACCESS_TOKEN=your_access_token
X_ACCESS_TOKEN_SECRET=your_access_token_secret

3. Activate (one command)

powershell -ExecutionPolicy Bypass -File scripts\setup-x-auto.ps1

This will: verify credentials → dry-run test → register Windows tasks (5 posts/day on your PC).

4. Cloud backup (PC off = still posting)

Push the same keys to GitHub Actions:

powershell -ExecutionPolicy Bypass -File scripts\setup-x-github-secrets.ps1

Workflow x-auto-post.yml runs every 30 min on GitHub servers.
