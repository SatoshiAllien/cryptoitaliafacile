1. Crea app su X Developer

  1. Vai su developer.x.com
  2. Crea un progetto e un'app con permessi Read and Write
  3. Genera API Key, API Secret, Access Token, Access Token Secret per l'account @TheRiser100x

2. Configura scripts/.env

X_API_KEY=your_api_key
X_API_SECRET=your_api_secret
X_ACCESS_TOKEN=your_access_token
X_ACCESS_TOKEN_SECRET=your_access_token_secret

3. GitHub Secrets (automazione cloud)

In GitHub → Settings → Secrets → Actions, aggiungi:

  • X_API_KEY
  • X_API_SECRET
  • X_ACCESS_TOKEN
  • X_ACCESS_TOKEN_SECRET

Il workflow x-auto-post.yml pubblicherà automaticamente 5 volte al giorno con immagine allegata.
