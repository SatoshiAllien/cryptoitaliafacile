Perché non vedi post su X

Errore 403: l'app X è in modalità solo lettura. Il bot può autenticarsi ma non può pubblicare tweet.

Fix in 5 passi

  1. Vai su developer.x.com → loggato come @TheRiser100x
  2. Apri la tua App → tab Settings
  3. Scorri a User authentication settings → clicca Set up o Edit
  4. Imposta:
    • App permissions: Read and Write
    • Type of App: Automated App or Web App
    • Callback URL: https://127.0.0.1
    Salva.
  5. Vai su Keys and tokens → Regenerate su Access Token and Secret → aggiorna scripts/.env

Test

python scripts\verify-x-credentials.py
python scripts\post-to-x.py --auto --slot 0 --no-image
