Perché non vedi post su X

Errore 403: l'app X è in modalità solo lettura. Il bot può autenticarsi ma non può pubblicare tweet.

Fix in 6 passi

  1. Vai su developer.x.com → loggato come @TheRiser100x
  2. Apri la tua App → tab Settings
  3. In alto, sezione App permissions → deve dire Read and write (non “Read”)
  4. User authentication settings → Edit → abilita OAuth 1.0a (non solo OAuth 2.0)
  5. Imposta:
    • App permissions: Read and write
    • Type of App: Automated App
    • Callback URL: https://127.0.0.1
    • Website URL: https://satoshiallien.github.io/cryptoitaliafacile/
    Clicca Save e attendi conferma.
  6. Keys and tokens → Regenerate Access Token + Secret → aggiorna scripts/.env

Piano API: se dopo il fix vedi ancora 403, il piano Free potrebbe non permettere post. Serve Basic (~$200/mese) su developer.x.com → Billing.
